=====YOU MUST HAVE UNTRADEABLE DROP NOTIFICATIONS ON FOR THIS PLUGIN TO WORK=====

With this plugin, you will be able to see the following things:
-   Your Current Abyssal Sire Kill Count (Begins tracking the first kill after installing)
-   The Last Kill Count You Received An Unsired (With a configuration option to set when the last KC was)
-   Current Dry Streak (Shows you how many kills since you've last had an Unsired)
-   Longest Dry Streak (The most number of kills between 2 Unsireds)
-   Most Spooned (The fewest number of kills between 2 Unsireds)

You can reset the following stats:
- All Stats (Sets you back to 0 in everything, including current KC)
- Last Unsired Kill Count
- Longest Dry
- Most Spooned

These stats can be reset by Shift + Right-Clicking the overlay on your screen and selecting which option you want to reset! <br>
